# Career Odyssey — 3D Portfolio

Career Odyssey is an immersive 3D personal portfolio that turns a traditional résumé into an interactive journey through space. Visitors pilot a spacecraft between planets to discover my background, technical skills, projects, experience, and contact information.

## Live Demo

Explore the portfolio at [leanhtuan.page.gd](http://leanhtuan.page.gd/?i=1).

## About the Project

This portfolio was created to present my journey as a Backend Developer in a more memorable and engaging way. Each planet represents a different part of my profile:

- **Origin** — personal introduction and career goals
- **Skills** — backend, frontend, database, security, system, and development tools
- **Projects** — selected applications and technical work
- **Experience** — education, activities, and development journey
- **Contact** — ways to connect with me

## Features

- Interactive 3D space environment built with Three.js and WebGL
- Flyable spacecraft with smooth keyboard and mouse controls
- Animated planets, orbital paths, particles, shaders, and visual effects
- Interactive information panels for every portfolio section
- Background music player and generated sound effects
- Responsive layout with joystick and action controls for mobile devices
- Full-screen, immersive user experience

## Controls

| Action | Control |
| --- | --- |
| Move forward | `W` |
| Move backward | `S` |
| Move left | `A` |
| Move right | `D` |
| Fire | `Space` |
| Navigate / look around | Mouse |

Mobile visitors can use the on-screen joystick and action button.

## Getting Started

### Prerequisites

- Node.js
- npm or Bun

### Installation

```bash
git clone https://github.com/bbi-yoykai1017/Portfolio.git
cd Portfolio
npm install
npm run dev
```

Open the local URL shown in the terminal to view the project.

### Production Build

```bash
npm run build
npm run preview
```
